마크다운(Markdown)은 텍스트에 서식을 입히는 아주 가볍고 직관적인 언어입니다. 
우리가 만든 Node.js SSG는 `marked` 라이브러리를 통해 마크다운을 완벽하게 지원합니다.

## 기본 텍스트 서식

여기는 일반 텍스트입니다. **이곳은 굵은 글씨**이고, *이곳은 기울임 꼴*이며, ~~이곳은 취소선~~입니다.

코드의 일부를 언급할 때는 백틱을 사용합니다: `console.log('Hello!')`

## 인용구 (Blockquotes)

다른 사람의 말이나 중요한 문장을 인용할 때는 이렇게 씁니다.

> "프로그래밍은 단순히 코드를 짜는 것이 아니라, 문제를 해결하는 논리적 사고의 과정이다."
> - 누군가 멋진 사람

## 리스트 (Lists)

### 순서가 없는 리스트
* 사과 🍎
* 바나나 🍌
  * 서브 아이템 1
  * 서브 아이템 2
* 오렌지 🍊

### 순서가 있는 리스트
1. Node.js 설치하기
2. 블로그 레포지토리 클론하기
3. `npm run dev` 실행하기

## 표 (Tables)

| 이름 | 역할 | 사용 라이브러리 |
|------|------|----------------|
| 마크다운 변환 | 텍스트를 HTML로 | `marked` |
| 메타데이터 파싱 | 머리말 추출 | `gray-matter` |
| 코드 하이라이팅 | 코드 색상 입히기 | `highlight.js` |

이제 이 문법들을 활용해서 멋진 글들을 작성해 보세요! 📝
